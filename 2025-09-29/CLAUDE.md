# Claude Code 設定

## プレゼンテーション作業

### スライド変更後の確認手順
スライドのデザインや内容を変更した後は、必ず以下の手順で確認してください：

1. **Playwright MCPでデザイン確認**
   - `mcp__playwright__browser_navigate` で該当スライドに移動
   - `mcp__playwright__browser_take_screenshot` でスクリーンショットを撮影
   - デザインの崩れ、レイアウトの問題、表示エラーがないかチェック

2. **確認するポイント**
   - 文字が切れていないか
   - HTMLタグが生のテキストで表示されていないか
   - レイアウトが崩れていないか
   - すべてのコンテンツがスライド内に収まっているか

3. **問題があった場合**
   - 即座に修正を行う
   - 再度確認手順を実行して修正を検証
